# Game List

**Game List** é um projeto desenvolvido como parte de um desafio proposto pelo professor Julio César da Escola DNC. para uma competição amistosa entre os alunos
onde exercitamos nossa criatividade e praticamos o que estamos aprendendo.

## Funcionalidades

- **Pesquisa por Nome:** Os usuários podem pesquisar jogos pelo nome para encontrar rapidamente os títulos que estão procurando.
- **Detalhes do Jogo:** É possível visualizar informações detalhadas sobre um jogo específico, incluindo descrição, classificação, plataforma e data de lançamento.
- **Comentários:** Os usuários podem adicionar comentários aos jogos, compartilhando suas opiniões e experiências com outros usuários.
- **Visualização de Comentários:** Além de adicionar comentários, os usuários também podem ver os comentários feitos por outros jogadores.

## Tecnologias Utilizadas

- **Frontend:** O frontend do aplicativo foi desenvolvido usando React, uma biblioteca popular de JavaScript para criar interfaces de usuário interativas.
- **Backend:** O backend é alimentado por Node.js e Express.js, que são frameworks populares para construir aplicativos web escaláveis e eficientes.
- **Banco de Dados:** Os dados dos jogos e comentários são armazenados no MongoDB, um banco de dados NoSQL altamente flexível e escalável.
- **API Documentation:** A documentação da API é fornecida usando Swagger, facilitando para outros desenvolvedores entenderem e usarem a API do aplicativo.
- **Estilo:** O aplicativo é estilizado usando Sass, um pré-processador CSS que oferece recursos poderosos para criar estilos atraentes e reutilizáveis.
- **Roteamento:** O React Router é usado para facilitar a navegação dentro do aplicativo, garantindo uma experiência de usuário suave e intuitiva.
- **Comunicação com o Servidor:** O Axios é usado para fazer solicitações HTTP entre o frontend e o backend, permitindo uma comunicação eficiente entre as duas partes do aplicativo.
- **ORM:** Mongoose é uma biblioteca de modelagem de objeto MongoDB para Node.js, tornando mais fácil trabalhar com MongoDB no aplicativo.
